## The core

**Python for data & scientific computing.** Roughly: Python fundamentals (types, control flow, functions, data structures) → **numpy** → **plotting (matplotlib)** → **pandas / data analysis** → a final stretch on modelling / **machine learning (unsupervised)**. It's a *doing* course — you're graded on writing and reading code, not reciting theory. The `ZF_InformatikII` summary is the compressed syllabus.

## Priority order

1. **Write code, don't just read it.** Redo every `uebung`/`exercise` from blank; run the notebooks.
2. **ZF InformatikII** (DE/EN) — the condensed reference; know every idiom in it cold.
3. **Cheat-sheets** (numpy / pandas / matplotlib) — the exam is fast; these turn recall into lookup.
4. **Past exams** from the inf collection — <https://lec.inf.ethz.ch/past_exams/> (I can scrape the Info-II ones on request). Drill them timed near the end.
